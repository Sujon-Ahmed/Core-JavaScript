Else if Statement
----------------------
To show a multi-way decision based on several conditions, We use else if Statement.
Syntax:-
    if(condition_1)
        {
            Statement_1_Block;
        }
    else if(condition_2)
        {
            Statement_2_Block;
        }
    else if(condition_n)
        {
            Statement_n_Block;
    else{
        Statement_x;
    }
