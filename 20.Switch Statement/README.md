Switch Statement
-------------------
Check several possible constant values for an expression.
Syntax:-
    Switch(expression)
        {
            case expression 1:
            {
                block of Statement 1;
                break;
            }
            case expression 2:
            {
                block of Statement 2;
                break;
            }
            default:
            {
                default value show.
                break;
            }
        }
